# Security Policy

RegIntel AI is published primarily as a portfolio and reference implementation.

Do not publish secrets, API keys, credentials, private endpoints, protected health information, or private datasets in this repository. Use `.env.example` as a template only, and keep real values in local environment files or a dedicated secret manager.

If you find a security issue, contact the maintainer directly through the contact channel listed on the maintainer's GitHub profile. Please do not disclose security issues publicly before the maintainer has had a reasonable opportunity to review them.
